Suponha que um dos itens tenha 2 e fator de encolhimento e os outros tres 1.
Formula: Unidade de encolhimento = espaco total - espaco dos itens / soma dos fatores de encolhimento dos itens.
Fica assim -> (500px - 800px) / 5 = -60px

Flex items com shrkink 1 : (-60px * 1) + 200px = 140px. Flex items com shrink 2 : (-60px * 2) + 200px = 80px.

flex-shrink com valor maior que zero define que Flex Itens devem encolher para caber no contêiner;
